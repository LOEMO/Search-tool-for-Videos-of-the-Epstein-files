# Before this tool can be used, you have to assure that the libraries pyautogui and pynput are already installed.

## If the libraries arent already installed 
-run 'py -m pip install pynput pyautogui' in your cmd for Windows<br>
-run 'pip install pyinput pyautogui' in your terminal for Linux

## Keybinds:
-'d' for next video<br>
-'a' for previous video<br>
-'m' to memorize a video in the database<br>

## Database
the database is called 'mem_links1.txt'. Here are all you memorized Links saved (In order and with timestamps)<br>
Your database will be saved, where the python file is saved.
